# Directori src

> Path absolut: /FONTS/src

## Descripció del directori
Aquest directori conté tot el codi del projecte organitzat per packages

## Elements del directori

- **Directori main:**
Conté tots els codis de les classes del model.

- **Directori test:**
Conté tots els tests unitaris, fets amb JUnit, de les classes del model conceptual classificats per les classes que realitzen alguna funcionalitat i les classes que defineixen tipus de dades.

- ***Makefile***
Permet executar el sistema i testejar les classes a partir dels drivers. Conté diverses opcions.

## Descripció Makefile

- `make all`

      Compila totes les classes del sistema

- `jarDriver`

      Executa els drivers del programa
