# Directori DOCS

> Path absolut: /DOCS

## Descripció del directori
Aquest directori conté tota la documentacio del projecte

## Elements del directori

- **Arxiu Diagrama_capa_persistencia_definitiu.pdf:**
Conté el diagrama de classes de la capa de persistència del projecte.

- **Arxiu Diagrama_capa_presentació_definitiu.pdf:**
Conté el diagrama de classes de la capa de presentació del projecte.

- **Arxiu Diagrama Casos de Uso.pdf:**
Conté el diagrama de casos d'ús del projecte.

- **Arxiu Diagrama_de_classes_definitiu.pdf:**
Conté el diagrama de classes de la capa de domini del projecte.

- **Arxiu Entrega 1.pdf:**
Conté la descripció del projecte, diagrames, classes, costos algorítmics i organització del treball.

- **Arxiu Entrega 2.pdf:**
Conté la descripció del projecte, diagrames de totes les capes,explicació d'aquestes i costos algorítmic.

- **Directori Doxygen:**
Conté els arxius corresponents per implementar el doxyFile
