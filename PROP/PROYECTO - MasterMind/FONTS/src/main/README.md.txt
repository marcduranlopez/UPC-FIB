# Directori main

> Path absolut: /FONTS/src/main

## Descripció del directori
Aquest directori conté el codi del sistema.

## Elements del directori

- **Directori data:**
Conté els codis de la capa de data.

- **Directori domain:**
Conté els codis de la capa de domini.

- **Directori presentation:**
Conté els codis de la capa de presentation.

- **Arxiu Main.java:**
Arxiu que executa el joc a partir de la capa de presentació
