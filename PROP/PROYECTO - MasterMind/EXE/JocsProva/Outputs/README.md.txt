# Directori Outputs

Path absolut: /EXE/JocsProva/Outputs

## Descripció del directori
Aquest directori conté els outputs dels diferents jocs de prova 

## Elements del directori

- **OutputCrearCarregarConsultarPerfil.txt:**
Output per crear -> carregar -> consulart perfil

- **OutputGuardarCarregarPartida.txt:**
Output per guardar -> carregar partida

- **OutputPartidaCodeBreaker.txt:**
Output per executar una partida en codeBreaker

- **OutputPartidaCodeMaker.txt:**
Output per executar una partida en codeMaker

- **OutputStats.txt:**
Output per consultar els stats
