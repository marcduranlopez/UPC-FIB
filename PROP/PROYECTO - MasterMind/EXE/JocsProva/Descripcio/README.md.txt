# Directori Descripció

Path absolut: /EXE/JocsProva/Descripcio

## Descripció del directori
Aquest directori conté la descripció dels diferents jocs de prova 

## Elements del directori

- **DescripcioInputPartida.txt:**
Descripció dels Input de la partida

- **DescripcioInputPartidaCB.txt:**
Descripció dels Input de la partida com a codeBreaker

- **DescripcioInputPartidaCM.txt:**
Descripció dels Input de la partida com a codeMaker

- **DescripcioInputPerfil.txt:**
Descripció dels Input dels perfils

- **DescripcioInputRanking.txt:**
Descripció dels Input dels rankings

- **DescripcioInputStats.txt:**
Descripció dels Input dels stats
