# Directori Inputs
Path absolut: /EXE/JocsProva/Inputs

## Descripció del directori
Aquest directori conté els inputs dels diferents jocs de prova 

## Elements del directori

- **InputCrearCarregarConsultarPerfil.txt:**
Input per crear -> carregar -> consulart perfil

- **InputGuardarCarregarPartida.txt:**
Input per guardar -> carregar partida

- **InputPartidaCodeBreaker.txt:**
Input per executar una partida en codeBreaker

- **InputPartidaCodeMaker.txt:**
Input per executar una partida en codeMaker

- **InputStats.txt:**
Input per consultar els stats
