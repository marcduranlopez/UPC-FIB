# Directori EXE

Path absolut: /EXE

## Descripció del directori
Aquest directori conté els jocs de prova i l'executable del driver

## Elements del directori

- **Directori JocsProva:**
Hi ha els jocs de prova dividits en tres subdirectoris: Inputs, Outputs i Descripció

- **Directori JocsProva_DEFINITIUS:**
Hi ha els jocs de prova dividits en tres subdirectoris: Inputs, Outputs i Descripció

- **Directori out:**
Conté els resultats de cada execució

- **Arxiu driverDomain.jar:**
Arxiu .jar del driverDomain

- **Arxiu gson-2.8.6.jar:**
Arxiu .jar de la llibreria gson

- **Arxiu Main.jar:**
Arxiu .jar resultant de la compilació per jugar el joc
