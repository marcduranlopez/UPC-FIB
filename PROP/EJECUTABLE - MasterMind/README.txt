## MASTERMIND V1.1 ##

- Description -

This is a version of the board game MasterMind programmed in java.


- Instalation -

1. Enter the folder /MasterMind/Java Installer
2. Execute and install "JavaSetup8u371"
3. Exectue and install "jdk-19.0.2_windows-x64_bin"


- How to play -

Execute the "MasterMind" shortcut file and read the "Manual_d'Usuari_cat.pdf" in /Manual
