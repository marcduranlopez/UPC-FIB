EVALUATOR: Plataforma de gestión de problemas y cursos de programación.

La plataforma EVALUATOR integra una colección de problemas de programación, un repositorio de sesiones compuestas por problemas de dicha colección, un conjunto de cursos formados por una o más sesiones, y un conjunto de usuarios registrados. Se introducen las clases conj_problemas, Problema, conj_sesiones, Sesion, conj_cursos, Curso, conj_usuarios y Usuario.
