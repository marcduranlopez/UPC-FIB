\doxysubsection{Descripción detallada}
Representa un problema de la plataforma. 

El problema se representa con un identificador. Algunas de las características de cada problema son, por ejemplo, el número {\itshape t} de envíos totales y el número {\itshape e} de envíos con éxito.
