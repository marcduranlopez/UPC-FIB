\doxysubsection{Descripción detallada}
Representa un conjunto de mensajes inrtroducidas en la plataforma. 

Cada mensajes del conjunto es representado con un identificador {\itshape idm}. Como prerrequisito no puede haber dos mensajes en el sistema con el mismo identificador.
