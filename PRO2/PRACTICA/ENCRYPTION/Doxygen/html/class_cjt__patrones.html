Descripción detallada

Representa un conjunto de patrones inrtroducidos en la plataforma.

Cada patron del conjunto es representado con un identificador (idp).

void Cjt_patrones::leer ( )

Añade los patrones idp introducidos en la entrada del programa a un vector VEC_patrones de clases Patron.

Precondición
La clase Cjt_patrones esta inicializada.
Postcondición
Se obtiene el vector VEC_patrones con todas las clases Patron como parametro implicito del Cjt_patrones. Para cada patron añadido se leen sus valores en preorden con marca -1.
